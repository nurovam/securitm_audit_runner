# Changelog

Все заметные изменения проекта фиксируются в этом файле.

## [Unreleased]

## [0.2.0] - 2026-04-14

### Added

- Единый backlog проекта и правила ведения репозитория.
- CLI entrypoint через `pyproject.toml`.
- Базовый набор unit-тестов и GitHub Actions workflow.
- Типизированный протокол контекста аудита.

### Changed

- PDF-экспорт стал ленивым и перестал валить запуск без `reportlab`.
- Поток интеграции с SecurITM стал устойчивее к сбоям API.
- Создание задач в SecurITM стало защищено от простых дублей.
- Перед созданием задачи агент сначала ищет уже существующую открытую задачу и не создаёт дубликат, если она ещё не закрыта.
- Проверка системного cron теперь проверяет файлы внутри `cron.*` каталогов.
- Плагин `met_rekom_linux` больше не использует shell для листинга каталогов.

### Documentation

- Добавлен English README.
- Добавлена документация по manual checks.
